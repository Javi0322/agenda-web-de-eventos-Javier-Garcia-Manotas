# Agenda de eventos del centro

Aplicación web desarrollada con **React y Vite** que permite consultar y organizar los eventos del centro educativo, como charlas, torneos, talleres y excursiones.

La aplicación permite buscar y filtrar eventos, ver el detalle de cada uno y marcar eventos como favoritos.

## Tecnologías utilizadas

- React
- Vite
- JavaScript
- CSS

## Cómo ejecutar el proyecto

### Requisitos previos

- Tener instalado **Node.js** y **npm**

### Pasos para ejecutar

1. Instalar las dependencias del proyecto:

```bash
   npm install
```

2. Ejecutar la aplicación:

```bash
   npm run dev
```

3. Abrir en el navegador la URL:
   http://localhost:5173/
